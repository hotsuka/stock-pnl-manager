## ドキュメントの更新

ドキュメントは常に最新の状態に保つよう心がけてください。

### 更新が必要な場合
- 新機能の追加時
- APIの変更時
- 重要なバグ修正時
- デプロイ手順の変更時

### 更新方法
1. 該当するドキュメントファイルを編集
2. 変更内容を明確に記述
3. 変更日時とバージョンを更新
4. Pull Requestを作成（チーム開発の場合）
